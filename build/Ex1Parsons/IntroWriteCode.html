Introduction to Write Code Problems¶

In write code problems you will be given the start of a function followed by hidden code to test the function and will be asked to finish writing the function as shown in the figure below.

Figure 1: What a Write Code Problem Looks Like When You Start

Start the Exam¶

Click the button to start the exam and show the problem.

Failing Tests¶

The hidden tests that are after the function will check that the function returns the correct values after you click the button. Each test will print if the test was successful or if it was not as shown in the figure below. If a test fails fix the code and run it again to test it again.

Figure 2: What a Write Code Problem Looks Like When Tests Fail

Passing Tests¶

If the code is correct the tests should all print as shown in the figure below.

Figure 3: What a Write Code Problem Looks Like When all Tests Pass

Finish the Exam¶

Click on the button below the question when you have either gotten the question correct or are ready to move on.
